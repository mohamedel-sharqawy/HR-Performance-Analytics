# HR Performance & Satisfaction Analysis
## Project Overview
This project focuses on analyzing Human Resources (HR) data to explore the relationship between employees' education level, job satisfaction, and performance rating. The goal is to identify patterns and insights that can help organizations better understand how these factors influence employee performance and overall workplace satisfaction.

The dataset includes several employee-related attributes such as Education Level, Performance Rating, Job Satisfaction Level, and Employee Rating Categories. By examining these variables, the project aims to answer important questions such as:

Does a higher education level lead to better employee performance?

Is there a relationship between employee satisfaction and performance ratings?

Which satisfaction levels are most common among high-performing employees?

Through data cleaning, exploratory data analysis (EDA), and visualization, this project will highlight trends and correlations within the HR dataset. The insights obtained can help HR departments make more informed decisions related to employee development, training programs, and workplace improvement strategies.

The analysis will be performed using data analysis tools and techniques to transform raw HR data into meaningful insights that support better organizational decision-making.

## Tools & Technologies

• Python – for data cleaning, manipulation, and analysis.  
• Pandas & NumPy – for handling datasets and performing numerical operations.  
• Matplotlib & Seaborn – for visualizing trends and correlations.  
• Jupyter Notebook – for interactive coding and presenting analysis step by step.  
• SQL – for querying and managing employee data stored in relational databases.  
• Power BI – for creating interactive dashboards and reports for stakeholders.  
• Tableau – for advanced data visualization and storytelling with HR metrics.  
• Git & GitHub – for version control and collaboration.  
• Statistics – for performing descriptive and inferential statistics on employee data.  
• Excel / CSV – as the format for raw HR datasets.
